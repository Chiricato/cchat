# cchat
Secure and lightweight chat for local networks

## How to install?
-------------------------------------------

```
$ npm install cchat -g
```

## How to use?
-------------------------------------------

To start select the port
```
$ sudo cchat set-port=PORT_NUMBER
```

And start 
```
$ cchat start
```

And browse your server
```
http://SERVER_IP:PORT_NUMBER // example - http://192.168.1.3:3030
```
